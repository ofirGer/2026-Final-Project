## שני פורמטים — מה ההבדל

### קבצי `.svg` — לשימוש מיידי
- ניתן לפתוח בכל דפדפן (לחיצה כפולה על הקובץ)
- ניתן להכניס ישירות ל-Word: **Insert → Pictures → This Device**
- וקטורי — איכות מושלמת בכל זום
- מוכן לשימוש כמו שהוא

### קבצי `.drawio` — לעריכה
- ניתן לפתוח ב-**app.diagrams.net** (חינם, בדפדפן — אין צורך להתקין)
- או בתוכנת **draw.io desktop**
- מאפשר לערוך טקסטים, להזיז רכיבים, לשנות צבעים
- מאפשר לייצא ל-PNG / JPEG / PDF איכותי

## איך לפתוח ולערוך ב-draw.io

1. גש ל-https://app.diagrams.net (בדפדפן, חינם)
2. בחר **Open Existing Diagram**
3. בחר **Device** ובחר את קובץ ה-`.drawio` הרצוי
4. ערוך כרצונך
5. ייצוא לתמונה: **File → Export As → PNG** (או SVG / PDF)

## איך להכניס לספר ה-Word

### דרך 1: SVG ישירות (מומלץ)
1. ב-Word, מקם את הסמן במקום שבו אמור להופיע התרשים
2. **Insert → Pictures → This Device**
3. בחר את קובץ ה-`.svg`
4. Word מציג את התמונה — היא וקטורית ונראית מצוין בכל גודל

### דרך 2: PNG מ-draw.io
1. פתח את קובץ ה-`.drawio` באתר draw.io
2. **File → Export As → PNG** (בחר רזולוציה גבוהה — לפחות 200% או 300 DPI)
3. שמור את ה-PNG
4. ב-Word: **Insert → Pictures → This Device** והכנס את ה-PNG

## טיפ: לראות איך נראה SVG

לחץ פעמיים על קובץ `.svg` — הוא ייפתח בדפדפן ברירת המחדל ותוכל לראות אותו מיד.
